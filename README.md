# Tip Calculator

A simple and interactive web application that helps users calculate the tip amount and total bill based on a given percentage. This project is built using HTML, CSS, and JavaScript and is useful for real-life scenarios like dining out or splitting bills.

## Features

- Enter bill amount
- Select tip percentage
- Calculate tip instantly
- Display total amount (bill + tip)
- Clean and user-friendly interface
- Responsive design (basic)
